Configuring a Static Website With S3 And CloudFront

Project Overview

Our project focuses on creating a cloud storage solution using AWS S3. We aim to enable users to upload files, share them securely with different access permissions, and use version control to track changes to the files.

We will configure a static website using Amazon S3 and integrate CloudFront for better content delivery. Our project also involves analyzing the case study of Dropbox, a leading cloud storage provider.

Case Study: Dropbox

Dropbox provides cloud storage to millions of users, offering services like file storage, sharing, and versioning. Their solution uses a combination of cloud infrastructure and storage services, ensuring secure file sharing with flexible access control. Understanding how Dropbox handles file versioning and security will be crucial in implementing our own cloud storage solution.
